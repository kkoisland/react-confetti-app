# React Confetti サンプルアプリ 設計書

## プロジェクト概要

このプロジェクトは、React Confetti ライブラリの解説本のためのサンプルアプリケーションです。
読者が本を読んで理解した後、実際のサンプルで「ああ、こういうことか」と確認できるように設計されています。

**執筆・実装の進め方**: 実装する → 章を書く → 次の章

## 技術仕様（簡潔版）

**技術スタック**:
- React 19.x + TypeScript 5.9.x + Vite
- Tailwind CSS（スタイリング）
- React Router v6（ルーティング）
- react-confetti（紙吹雪ライブラリ）
- Biome（リンター/フォーマッター）

**ルート構造**:
```
/                    → ランディング/概要ページ（オプション）
/basic               → 第2章 基本の紙吹雪
/countdown           → 第3章 カウントダウンとお祝い
/toast               → 第4章 タスク完了の演出
/seasonal            → 第5章 季節のエフェクト
/playground          → 第6章 パラメータの探求
```

**デザイン方針**:
- モダンで美しいUI（グラデーション背景、ガラスモーフィズム）
- 上部ナビゲーション + メインエリア（紙吹雪を最大限広く見せる）
- ダーク/ライトモード対応
- レスポンシブデザイン

**レスポンシブ対応の方針**:
1. 基本レイアウト（上部ナビなど）：最初からレスポンシブ対応
2. 各ページ（/basic, /countdown等）：デスクトップ優先で実装、見た目が揃ってから調整

## 付録A：環境構築手順

**注意**: 実装時の参考として詳細な手順を記載していますが、本に掲載する際は大幅に削除して「補足セットアップ手順」にする可能性があります。理由：
- 初心者には詳しく書いても分からない
- 経験者には公式ドキュメントの方が有用
- 環境の変化ですぐ古くなる
- 誰にも参考にならない可能性が高い

### 完全セットアップ手順（実装時の参考用）

#### 1. Tailwind CSSのインストールと設定
```bash
pnpm add -D tailwindcss postcss autoprefixer
pnpm exec tailwindcss init -p
```

#### 2. React Routerのインストールと設定
```bash
pnpm add react-router-dom
```

#### 3. react-confettiのインストール
```bash
pnpm add react-confetti
pnpm add -D @types/react-confetti
```

#### 4. 基本レイアウト（上部ナビ + ルーティング）

**重要**: この段階でレスポンシブ対応を含めて実装します。各ページ（/basic等）はデスクトップ優先で実装するため、レイアウトだけはここでしっかり作ります。

プロンプト例：
```
React Router v6を使って、以下の構成のアプリを作成してください：

- 上部ナビゲーションバー（/basic, /countdown, /toast, /seasonal, /playground）
- メインエリア（各ページのコンテンツ、画面全体を使える）
- Tailwindでモダンなレイアウト
- ダーク/ライトモード切り替え機能（右上にトグルボタン）
- レスポンシブデザイン（モバイルではハンバーガーメニュー、デスクトップでは横並びメニュー）
- 紙吹雪が画面全体に広がるよう、メインエリアは最大限の広さを確保
```

#### 5. おまけ：ダークモード切り替え
サンプルアプリをより見やすくするため、ダーク/ライトモード切り替え機能を実装します。
これはReact Confettiの機能ではなく、アプリ全体のUI改善のためのおまけ機能です。

**実装のポイント**：
- Tailwindの `dark:` クラスを使用
- 上部ナビの右上に切り替えボタンを配置
- `localStorage` で設定を保存（オプション）

本書では詳細な説明は省略しますが、実装されたサンプルコードを参考にしてください。
